# Bergère Enchantée

Website for upholstery, using pnpm & SvelteKit.

See on https://bergereenchantee.fr.

Hooked to https://huggingface.co/spaces/coyotte508/bergere-ci-cd

## Install

```
pnpm install
```

## Run

```
pnpm dev
```
